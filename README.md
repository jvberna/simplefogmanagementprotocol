# simplefogmanagementprotocol
Simple FOG management protocol prototype applied to IoT infrastructures

Install libraries
- npm i 

Execute components in order from package.json:
- 1 balancer_main
- 2 balancer_subs
- 3 coordinator_main
- 4 coordinator_subs
- 5 processor1
- 6 processor2
- 7 processor3
